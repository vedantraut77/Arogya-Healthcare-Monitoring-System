# 🌐 Arogya Web Application

The **Arogya Web Application** is the central interface of the Arogya Healthcare Monitoring System. It provides healthcare professionals with a modern dashboard to remotely monitor patient health, visualize real-time sensor data, manage patient records, and receive alerts from connected IoT devices.

This application is built using **React**, **Firebase**, and modern web technologies, enabling secure authentication and real-time cloud synchronization.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Arogya-Healthcare-Monitoring-System.git
```

Go to the website directory

```bash
cd website
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

---

# 🔐 Firebase Authentication Setup

This project uses **Firebase Authentication**.

Create your own Firebase project and configure the Firebase credentials inside:

```
src/config/firebase.js
```

or through your environment configuration.
